There's a whole lot to love about HTML5. It's a pragmatic language, designed to support the way we actually make websites, and to make everyone's life easier.

  • Simplified doctype, script and link, much nicer.
  • New semantic tags.
  • New media tags for embedded video and audio.
  • Improved forms.
  • Optional self closing elements.
  • Looser validation. We can wrap more things inside other things than before, in a good way.

The HTML5 Logo

When we talk about HTML5, we are generally talking about more than just the markup. Although technically not a part of HTML5, we usually also mean things like:

  • Geocoding - to find the location of a device.
  • Web Sockets - an AJAX replacement.
  • Local Storage - improved cookies.
  • Web Workers - multithreaded JavaScript
  • SVG, Canvas, CSS3 3D and WebGL - embedded graphics
  • CSS3 - improved rendering.

The spec is constantly evolving, and probably won't be close to final until 2012, if ever. This doesn't mean we can't start to use it though.
